1. 
Name: FinishTask


2. 
Description: Sets a task to "Done" in a specified tasklist.


3. 
Parameters:


a. 
ID
     The ID of the tasklist.


b. 
Task ID
     The zero-based index of the task that should be set to done.
 


4. 
Examples:
```json
FinishTask:[0|5];
```

None
