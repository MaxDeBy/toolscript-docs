# HideEvidence (HE)
---
### Description
This instruction hides the evidence thumbnail in the top left corner.

### Parameters
This instruction has no parameters.

### Examples:
#### Example #1: Hiding the currently displayed evidence thumbnail.
```
1:  HideEvidence;
```
