# FinishInvestigation
---
- **Name:** FinishInvestigation (FinINV)
- **Description:** Finishes an investigation
- **Parameters:**
  - **Investigation ID:**  
    The ID of the investigation that should be finished.
 
- Examples:
```
1:  FinishInvestigation:["FinalInvestigation"];
2:  FinINV:[1];
```

> None
