1. Name: RevealTopic

2. Description: Reveals a talking topic in an investigation.

3. Parameters

    a. Investigation ID
    
    The ID of the investigation containing the talking topic.
    
    b. Index of the move option
    
    The 1-based index of the talking topic.
4. Examples:
```json
RevealTopic:[0,1];
```
