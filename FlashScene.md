# FlashScene
---
- Name: FlashScene (Flash)
- Description: Flashes the screen white.
- Parameters:
  - **Duration:**  
    Specifies how long the flash takes. Default is 200ms.

- Examples:
```
1:  FlashScene:[200];
2:  Flash:[100];
```

- Remarks:
> None
