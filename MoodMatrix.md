# PlayMoodMatrix
---
- **Name:** PlayMoodMatrix (PlayMM)
- **Description:** Starts a mood matrix sequence.
- **Parameters:**
  - **Mood Matrix ID:**  
    The Id of the Mood Matrix sequence to be played

- Examples:
```
1:  PlayMoodMatrix:["AngryMaya"];
2:  PlayMM:[2];
```

- Remarks:
> None
