1. Name: HideTopic

2. Description: Hides a talking topic in an investigation.

3. Parameters

    a. Investigation ID
    
    The ID of the investigation containing the talking topic.
    
    b. Index of the move option
    
    The 1-based index of the talking topic.
4. Examples:
```json
HideTopic:[0,1];
```

>None
